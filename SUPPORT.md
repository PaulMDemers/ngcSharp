# Support

ngcSharp is an experimental emulator project. There is no end-user support channel for playing commercial games yet.

Use issues for:

- Build failures.
- Test failures.
- Reproducible crashes.
- Incorrect emulator behavior with a small homebrew or synthetic repro.
- Documentation gaps.

Please do not open issues asking for:

- Game downloads.
- BIOS/IPL files.
- Nintendo SDK files.
- Help bypassing copy protection or platform restrictions.
- General "does game X work?" requests without logs or a repro command.

For compatibility findings, include:

- The exact command.
- Instruction budget.
- Any generated trace or artifact path.
- Whether the image is ISO, GCM, or RVZ.
- The game ID if available.
