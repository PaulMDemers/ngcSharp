# Code Of Conduct

This project should be a calm, technical place to build an emulator together.

Expected behavior:

- Be respectful and specific.
- Assume good intent, but accept corrections quickly.
- Keep reviews focused on code, tests, docs, and emulator behavior.
- Credit research sources and prior work.
- Respect legal boundaries around copyrighted assets and proprietary SDK material.

Unacceptable behavior:

- Harassment, insults, or personal attacks.
- Posting copyrighted game images, BIOS/IPL dumps, proprietary SDK files, or links to them.
- Pressuring contributors to provide illegal assets.
- Repeatedly derailing technical issues.

Maintainers may edit, hide, or remove comments and contributions that violate these expectations.
